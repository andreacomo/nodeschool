```

```

## test-anything ##

```

```

## learnyoumongo ##

Start Mongo with:

```
mongod --dbpath <path/to/data/folder>
```

Run a program with

```
learnyoumongo run path/to/program.js

Test a program with

```
learnyoumongo verify path/to/program.js

Tested on MongoDb 3.0.6
